= TinyMCE Extended =

TinyMCE with configuration options.

== Features ==

- Support TinyMCE 3.2.5 
- Packed with Image Manager (http://dustweb.ru/projects/tinymce_images/)
- Select available plugins
- Customizable menu's
- extendable valid_elements (will also be excluded from elgg input sanitizer)
